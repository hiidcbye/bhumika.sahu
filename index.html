Project 01 · AI · Finance · Web App

Saavy
AI Finance Copilot

An intelligent student finance web application that detects spending anomalies, forecasts trends, and delivers personalized investment guidance through a conversational AI copilot.

The Problem

Students managing finances for the first time have no early warning system for overspending or unusual patterns. Generic budgeting apps don't speak to student spending behaviour — they just show numbers.

The Solution

Saavy uses rolling averages and Z-score deviation analysis to flag anomalies in real time. A conversational LLM-powered finance copilot then gives students actionable, personalized guidance.

How it was built

  • React frontend with responsive workflows for seamless financial analysis
  • FastAPI backend powering scalable REST APIs with PostgreSQL for persistent data
  • Pandas-based forecasting engine: rolling mean baseline → Z-score deviation → anomaly flag
  • LLM API integration for natural-language finance copilot with investment recommendations

Project 02 · AI · Freelance · Risk Detection

ScopeGuard
Scope Creep Detector

Detects freelance project scope creep, calculates unpaid revenue risk, and auto-generates professional client responses — powered by Gemini API prompt pipelines.

The Problem

Freelancers lose thousands to invisible scope creep — clients add work, deadlines extend, but rates never change. There was no tool to quantify the damage or respond professionally.

The Solution

ScopeGuard runs project scope + extra-work logs through a structured Gemini API prompt pipeline and outputs: a risk score, financial impact estimate, and a client-ready message.

How it was built

  • Structured Gemini API prompt pipelines for multi-step financial risk analysis
  • React dashboard with real-time risk gauge and AI-generated response panel
  • Node.js backend with secure REST APIs and prompt-injection safeguards
  • Deployed live on Render — fully production-ready
